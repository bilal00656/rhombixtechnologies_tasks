# Personal Portfolio Website

This project is a responsive Personal Portfolio Website developed as part of an internship task.

## 📌 Project Overview
The portfolio showcases my skills, projects, and contact information in a clean and user-friendly interface.

## 🚀 Features
- Responsive design (works on mobile, tablet, desktop)
- Projects showcase section
- Contact form using PHP
- Smooth navigation and UI
- Clean and modern layout

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- PHP

## 💻 Development Tools
- Visual Studio Code
- XAMPP (for running PHP locally)

## 📂 How to Run the Project

1. Install XAMPP
2. Move project folder to:  C:\xampp\htdocs\

3. Start Apache server from XAMPP
4. Open browser and go to:  http://localhost/your-folder-name/
